Movie Showcase Website

A simple movie poster gallery designed using HTML and CSS Flexbox. The website includes different genres such as Action, Comedy, and Romance, each displayed in a clean visual layout.

Features:
• Categorized movie sections
• Poster grid alignment using Flexbox
• Selected posters link to IMDb for additional info
• Easy-to-understand and clean UI

Technologies Used:
• HTML
• CSS (Flexbox for layout)
